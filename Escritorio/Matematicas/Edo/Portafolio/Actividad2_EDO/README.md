# Solución de Portafolio EDO

Estructura generada modularmente por apartados:

- `lib/`: Algoritmos numéricos puros (Euler, RK, Taylor).
- [cite_start]`core/`: Funciones aisladas para cada apartado (A, B, C, D)[cite: 5].
- `scripts/`: Archivos principales `main_problema1.m` y `main_problema2.m` que resuelven la actividad.

## Ejecución
1. Abrir MATLAB.
2. Entrar en la carpeta `scripts/`.
3. Ejecutar `main_problema1` o `main_problema2`.
