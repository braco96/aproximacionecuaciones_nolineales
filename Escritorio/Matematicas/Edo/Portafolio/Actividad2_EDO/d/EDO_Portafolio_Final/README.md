# Portafolio EDO - Estructura Modular

Proyecto generado automáticamente con trazabilidad completa.

## Estructura
* **`lib/`**: Motores matemáticos puros (Euler, RK4, etc).
* **`core/`**: Gestores de lógica (Analítico, Numérico, Gráfico, Errores).
* **`scripts/`**: Archivos ejecutables `main_problema1.m` y `main_problema2.m`.

## Ejecución
1. Abrir MATLAB.
2. Navegar a la carpeta `scripts/` dentro de este proyecto.
3. Ejecutar `main_problema1` o `main_problema2`.
   (Las rutas se configuran automáticamente).
